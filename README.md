# 서울시 범죄 데이터 · CCTV 설치 데이터 분석 프로젝트

이 저장소는 서울시의 범죄 발생/검거 데이터, 일반 CCTV/지능형 CCTV 설치현황, 
그리고 유흥업소 인허가 데이터를 활용하여 다음 4가지 가설을 검증하는 프로젝트입니다.

## 📌 사용된 데이터 (CSV)
- 서울시 5대범죄 발생현황 통계(2020~2024).csv  
- 서울시 자치구 (연도별) CCTV 설치현황.csv  
- 서울시 자치구 지능형 CCTV 설치현황.csv  
- 서울시 유흥주점영업 인허가 정보.csv  

## 📌 분석 내용 요약
1. CCTV 설치 증가가 검거율 증가로 이어지는가?
2. 유흥업소 증감이 범죄 증가와 관련 있는가?
3. 유흥업소 증감이 CCTV 효과를 억제하는 혼란변수인가?
4. 지능형 CCTV가 일반 CCTV보다 효과가 좋은가?

## 📌 주요 소스코드
- final.ipynb : 전체 전처리/분석 함수 및 main 실행 구조 포함

## 📌 실행 환경
- Python 3.x
- pandas, numpy, seaborn, matplotlib
